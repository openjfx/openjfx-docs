Run HelloWorld using Gradle

Similar to Maven, we can declare the required JavaFX modules in the build.gradle file. However, for Gradle we need to specify the JavaFX gradle plugin: 


plugins {
  id 'application'
  id 'org.openjfx.javafxplugin' version '0.0.5'
}

Next, we add the required modules. For instance, if we only need the javafx.controls module, we will include: 


javafx {
    modules = [ 'javafx.controls' ]
}

Run the application (e.g. use HelloFX.java from the given sample) using:

  • Linux/Mac
  • Windows 

./gradlew run


gradlew run
Note: Make sure to use gradle version 4.10+ and set the JAVA_HOME environment variable to JDK 11.
